# TicTacToe
A classic two-player tic-tac-toe game with turn logic and automatic win/draw detection.

## Features
- Local 2-player mode  
- Win, loss, draw detection  
- Grid reset button  
- Clean and intuitive interface  

## How to Run
1. Import `.aia` file into MIT App Inventor.
2. Connect device/emulator.
3. Test with **AI Companion**.

## Tech Stack
- MIT App Inventor  
- Button & Layout components
